Architecture

Dual Detection Architecture

Three parallel AI modules work in concert and fuse into a unified threat prevention pipeline.

LZ-JC-DBSCAN

Unsupervised Clustering

Detects novel anomalies and zero-day attacks using Lempel-Ziv complexity and Jaccard similarity.

EPRC-RPOA

Feature Optimization

Entropy-based pruning with recursive optimization keeps the most discriminative flow attributes.

EG-GELU-GRU

Deep Classification

GRU-based sequence modeling with GELU activation tracks temporal attack behavior in real time.

Setup

System Requirements

Runs efficiently on standard hardware. No GPU required for inference.

Hardware Requirements

  • Processor: Intel i5 / AMD Ryzen 5 or equivalent (multi-core)
  • RAM: 8-16 GB recommended
  • Storage: 50-100 GB for datasets, PCAP files, and models

Software Requirements

  • Operating System: Windows 10/11 laptop or desktop
  • Runtime: Standalone Windows EXE, no Python installation required
  • Network Tools: Built-in live capture (Scapy + CICFlowMeter pipeline) and offline PCAP analysis are both supported
  • Libraries: Model, UI assets, and Python dependencies are bundled into the packaged release
